# AfuseKtV-release


<a href="README-EG.md" >English</a> <br>
欢迎大家使用Afusekt TV版本.<br>
本库为该App发布release版本，不包含任何代码.<br>
包含功能刮削，播放视频.<br>
支持协议：Alist，SMB，Webdav，Emby（直连），jellyfin（直连），阿里网盘.<br>
资源库仅支持：通过AfuseKt移动端同步资源库.<br>

最低版本支持安卓6.<br>
